# System
W tym folderze znajdują się techniczne elementy potrzebne do korzystania z PSS. Składają się na to:
- plugin
- słowniki

## Plugin
Plugin plover-polish-pss zawiera w sobie plugin do programu Plover. Obsługuje następujące protokoły:
- klawiatura
- Gemini PR

### Instalacja
W przyszłości plugin zostanie opublikowany w repozytorium programu Plover. Tymczasowo konieczna jest ręczna instalacja.
1. Należy pobrać cały katalog plover-polish-pss
2. Należy wkleić go do katalogu zawierającego program Plover
3. Uruchomić w tym miejscu linię komend z uprawnieniami administratora
4. W linii komend podać następujące polecenie: `.\plover_console.exe -s plover_plugins install .\plover-polish-pss`

Uwaga - instalację przeprowadza się na własną odpowiedzialność - pomimo podjęcia najlepszych starań, aby plugin nie zawierał błędów,
autor nie gwarantuje poprawności i nie bierze odpowiedzialności za korzystanie z pluginu.
